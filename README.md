# markdown-test
This project aims to provide a practical introduction to the use of Markdown, a lightweight markup language that allows for simple and effective text formatting. Through a series of exercises and examples, participants will learn to create well-structured documents using headers, lists, links, and images.
## What is Markdown?

Markdown is a simple way to style text on the web. It uses plain text formatting syntax, which means you can write your content in a way that is easy to read and write, and then convert it into HTML for web pages.

Basic Syntax:

Headings: You can create headings by using the # symbol. The number of # symbols indicates the level of the heading.

# Heading 1
## Heading 2
### Heading 3

## Bold and Italics:

To make text bold, wrap it in double asterisks: **bold text**.
To make text italic, wrap it in single asterisks: *italic text*.
Lists:

Unordered lists can be created using asterisks, plus signs, or hyphens:
* Item 1
* Item 2
Ordered lists use numbers followed by a period:
1. First item
2. Second item
Links: You can create hyperlinks by wrapping the link text in brackets and the URL in parentheses:

[OpenAI](https://www.openai.com)
Images: Similar to links, but with an exclamation mark in front:

![Alt text](image-url.jpg)
Code: For inline code, wrap the text in backticks:
`code`
For code blocks, use triple backticks:
```
code block
```
